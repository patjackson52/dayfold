Drop a note, get a researched card back

A member’s free-form note is treated as a bounded remote command to a key-holding Claude loop. On a later run the loop researches the note and authors a result card into the right hub — it does not reply, and it is not a chat. The arc is capture → “Dayfold is working on this…” → an AI-authored card with provenance. It may fetch external info to fulfil the command, and it acts only within what you can already see.

One command, end to end

The composer is one open field, but a note is a bounded command to the AI loop — not a chat turn.
